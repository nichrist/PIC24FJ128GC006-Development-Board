#
This is an evaluation & development board for PIC 24FJ128GC006 microcontroller. The two layer PCB board dimensions are 10x10cm. It consist of a 3V3 and 5V -5V regulators, pin headers connected to each one of the PIC’s port IO pins, USB type B connector, RS232 & RS485 transceivers, 3V3 / 5V logic level translators, an ICSP (in circuit serial programming) connector, switches and LEDs (4), temperature sensor, OpAmp ICs.
